java反射

发表于 2018-09-26 | 分类于 JavaSE

JAVA反射机制是在运行状态中，对于任意一个类，都能知道这个类的所有属性和方法：对于任意一个类都能够调用它的任意一个方法和属性；

获取Class对象的三种方式

Student stu = new Student();//产生一个Student对象，一个Class对象1.Class stuClass = stu.getClass();
2.Class stuClass = Student.class;
3.Class stuClass = Class.forname(包名.类名);
在运行期间一个类只能产生一个Class对象
